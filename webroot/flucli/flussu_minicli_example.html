Flussu MiniCli

Widget inline minimale per integrare un workflow Flussu in qualsiasi pagina web.

Esempio 1 — Integrazione minima

Un singolo <div class="flussu-minicli"> con le impostazioni dello script tag:

<div class="flussu-minicli"></div>

Esempio 2 — Righe iniziali personalizzate

Area risposta di 4 righe e input di 2 righe fin da subito:

<div class="flussu-minicli" data-response-rows="4" data-input-rows="2"></div>

Esempio 3 — Cambio lingua

Clicca un bottone per cambiare lingua al widget. Il placeholder e i messaggi si aggiornano.

// JavaScript — cambio lingua runtime
var widget = document.getElementById('minicli-lang-demo')._flussuMC;
widget.setLang('en');  // cambia a inglese

API JavaScript

// Accedi all'istanza del widget
var widget = document.querySelector('.flussu-minicli')._flussuMC;

// Cambia lingua runtime
widget.setLang('fr');  // it | en | fr | de | es

Note

  • Nessuna richiesta al server fino al primo invio dell'utente.
  • Le aree si espandono automaticamente col contenuto e non si rimpiccioliscono.
  • Il testo della risposta è selezionabile e copiabile.
  • La sessione è salvata in un cookie separato per ogni WID.
  • Invio con Enter. Nuova riga con Shift+Enter.
  • La lingua viene rilevata automaticamente da <html lang="..."> se non specificata.
